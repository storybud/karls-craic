What is going on here?

This is my website I've created to try and improve, in more depth, my understanding of the web and development

I will be experimenting everything from servers to frameworks, mainly around the frontend realm of development but sometimes other things like, processes, soft skills or a ardwino project.

What we've got so far

I've really only started so not much going on...

  • This is a static website, so far just html and css
  • I have deployed it on to a AWS S3 bucket with static web hosting enabled - currently no domain assigned
  • I've a small npm package that will run my build(?) scripts, this currently compiles my sass. Current packages are node-sass, node-sass-glob-importer and nodemon for watch.

Immediate tasks to complete.

  • Develop a custom advanced grid
  • Set up a Javascript bundler in the package json, maybe browsify, rollup or whats seem to be the in thing, webpack
  • Decide a module pattern to follow, seems like native ES6 modules is the way
  • Set up a deployment process from local to S3 bucket
  • Set up a github repo
  • Chose a nice font
  • Install a Javascript Templating Engine - i.e. Nunjucks or Engine examples
